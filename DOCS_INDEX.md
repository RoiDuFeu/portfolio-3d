## 💡 Contributing to Docs

Docs are part of the project! When you add a feature :

1. **Update README** (if user-facing)
2. **Log in CHANGELOG**
3. **Update TODO** (remove completed)
4. **Add to TESTING** (if testable)
5. **Cross-link** (reference other docs)

See [CONTRIBUTING.md](./CONTRIBUTING.md) for details.
