## 🎯 Les Planètes

### 🌱 Fertiscale (gauche)
Projet agri-tech. La planète **verdit** au fil de sa timeline.  
**Tech:** React Native, Python, Geo

### ⭐ godsPlan (centre)  
App églises Paris. Planète dorée élégante.  
**Tech:** Web scraping, React

### 🎵 Le Syndrome (droite)
Album piano. Planète **audio reactive** qui pulse !  
**Tech:** Web Audio API, particules 3D

## 🎬 Demo Flow

1. **Load** → Galaxie apparaît
2. **Scroll** → Voyage entre planètes
3. **Hover** → Planète brille
4. **Click** → Zoom + infos projet
5. **Explore** → Timeline, audio reactive

**Duration** : 2-5 min exploration typique

---

## 💡 Tips

- **Scroll horizontal** = voyage spatial
- **← →** = navigation clavier (fallback)
- **Mouvement souris** = parallax étoiles (subtil)
- **Audio** : ajoute `public/audio/lesyndrome.mp3` pour audio reactive
- **Textures** : custom assets dans `public/textures/` (Phase 2)
