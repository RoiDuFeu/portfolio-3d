SHADER PIPELINE:

Surface:
  Input:  5 textures + light direction
  ├─ Normal mapping (perturbNormal2Arb)
  ├─ Diffuse lighting
  ├─ Specular reflections (ocean)
  ├─ Cloud shadow subtraction
  └─ Night lights blend
  Output: vec4(finalColor, 1.0)

Clouds:
  Input:  Cloud texture + light direction
  ├─ Alpha transparency
  └─ Diffuse lighting
  Output: vec4(cloudColor * lighting, alpha)

Atmosphere:
  Input:  Light direction + camera position
  ├─ Edge detection (Fresnel)
  ├─ Directional fade
  └─ Smooth falloff
  Output: vec4(blueColor, edgeIntensity)

FEATURES IMPLÉMENTÉES:

✅ Tangent-space normal mapping (dFdx/dFdy TBN matrix)
✅ Phong specular with warm sun color (1.0, 0.8, 0.3)
✅ Cloud shadow subtraction on specular strength
✅ Smooth day/night transition (smoothstep -0.05 → 0.05)
✅ Night city lights in shadowed areas
✅ Semi-transparent cloud layer with diffuse lighting
✅ Blue atmospheric rim glow (additive blending)
✅ Performance optimized (useMemo, 64×64 spheres)
✅ TypeScript strict mode compatible
✅ R3F conventions (shaderMaterial, auto-injected uniforms)
